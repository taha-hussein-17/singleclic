# E-Commerce Product Page

## Task Overview
A product listing & detail page using React + Tailwind CSS + FakeStore API.

## Features
- Sort by price & latest
- Category filter & price slider
- Zoom image on hover
- Responsive sidebar
- Loading state on "Add to Cart"

## Tech Stack
- React
- Tailwind CSS
- React Router

## Run Locally
```bash
npm install
npm start
